Frequently Asked Questions

What does the table show?

The table displays information about mice in the database. It includes details such as mouse ID, date of birth, sex, status, cage information, breeding details, and more.

How do I filter the table?

The Filter tab allows users to filter and search for specific data within a dataset based on various criteria.

  1. Select a table from the dropdown menu.
  2. Choose specific criteria such as Mouse ID, Sex, and Mouse Status for filtering.
  3. Click the "View Table" button to generate a table based on your selections.
  4. Optionally, you can further refine your search by entering values in the Mouse ID, Parent Cage ID, Cage Number ID, and Breeding Cage ID fields.
  5. Choose sorting options from the dropdown menus.
  6. Click the "Apply filters" button to apply your filters and sorting preferences.
  7. Click the "Reset filters" button to clear all filters and reset the table to its original state.

The generated table displays data entries that match the selected criteria and are sorted according to the chosen sorting option.

What is the breeding log tab used for?

The Breeding Log tab is used to log and manage breeding information such as cage IDs, setup dates, strains, and breeding statuses.

  1. Enter specific criteria such as Breeding Cage ID, Setup Date, Strain, Stud ID, Dam ID, Comments, and Breeding Status for filtering.
  2. Choose sorting options from the dropdown menu.
  3. Click the "Apply filters" button to generate a log based on your selections.
  4. Click the "Reset filters" button to clear all filters and reset the log to its original state.

The generated log displays breeding records that match the selected criteria and are sorted according to the chosen sorting option.

What is the Graphing tab used for?

The Graphing tab is used to visualize mouse frequencies based on different variables such as sex, mouse type, and cage number.

  1. Select a table option from the dropdown menu.
  2. Choose filtering criteria such as mouse status (alive or all) and grouping variables (Grouping Variable 1 and Grouping Variable 2).
  3. Select a chart type (Bar Plot or Pie Chart).
  4. Click the "Generate Chart" button to generate a chart based on your selections.
  5. Optionally, you can click the "Reset Filters" button to clear all selections and reset the chart to its default state.

The generated chart visualizes mouse frequencies based on the selected criteria and chart type.

How do I download the table and/or plots?

You can download the table as a CSV (Comma Separated Values) file by clicking on the "Download" button provided in the table interface. If you have created plots or graphs using external tools, you can usually export them as image files or in other suitable formats for download.

How do I update the table with additional mouse data?

To update the table with additional mouse data, you need to have appropriate access permissions. You can then use the excel upload to upload your own excel sheet. Make sure to download the instructions sheet and a template if you are not sure. First choose a table you want to upload into then upload your excel sheet.
